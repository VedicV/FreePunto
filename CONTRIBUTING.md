# Як долучитися

Дякую за інтерес до FreePunto. Проєкт маленький і прагматичний: macOS-only, ручні перетворення тексту, без автокорекції, телеметрії та фонової історії введення.

## Локальний запуск

```bash
swift build
swift test
swift run FreePunto
```

Якщо активні Command Line Tools не містять робочий XCTest, використовуйте Xcode toolchain:

```bash
DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test
```

## Збирання застосунку

```bash
./scripts/build_app.sh
open dist/FreePunto.app
```

## Збирання DMG

```bash
./scripts/build_dmg.sh
open dist/FreePunto-0.1.1.dmg
```

## Правила змін

- Зберігайте фокус першої версії: тільки macOS, тільки ручні команди.
- Не додавайте автокорекцію, словники, телеметрію, історію введення або мережеву обробку тексту.
- Для змін у `PuntoCore` додавайте або оновлюйте unit-тести.
- Для змін у AppKit-частині перевіряйте поведінку вручну в реальних текстових полях macOS.
- Коментарі й документацію пишіть українською. Користувацькі рядки російської та англійської локалізації залишайте відповідними мовами.

## Pull request

Перед PR перевірте:

```bash
swift test
```

У PR коротко опишіть:

- що змінено;
- чому це потрібно;
- як перевірено;
- чи змінюється поведінка з дозволами Accessibility або Input Monitoring.
