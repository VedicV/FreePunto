# Changelog

## v0.1.1

- Підготовлено публічний open source реліз.
- Документацію і коментарі коду переведено українською.
- Додано MIT License.
- Додано пряме посилання на DMG через GitHub Releases.
- Мова інтерфейсу за замовчуванням береться із системи для `uk`, `ru`, `en`; для інших мов використовується українська.
- Розширено `.gitignore` для SwiftPM, Xcode, локальних збірок і редакторних файлів.

## v0.1.0

- Перша робоча версія macOS menu bar застосунку.
- Перетворення розкладки EN/RU/UA.
- Перетворення виділеного тексту або останнього слова.
- Одиночний `Control` як основна гаряча клавіша.
- Регістр речення, нижній регістр, title case і виправлення Caps Lock.
- Фонетична транслітерація між кирилицею та латиницею.
- Локальне збирання `.app` і `.dmg`.
