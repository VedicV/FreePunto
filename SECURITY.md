# Безпека

FreePunto працює з користувацьким текстом локально і тільки після явної команди користувача.

## Що вважається security/privacy проблемою

- Надсилання тексту в мережу.
- Збереження початкового або перетвореного тексту на диск.
- Логування вмісту буфера обміну.
- Збереження історії введення.
- Обхід системних обмежень macOS для захищених полів.
- Неправильне відновлення буфера обміну після copy/paste fallback.

## Як повідомити

Поки окремої security-пошти немає, створіть приватний контакт через профіль власника репозиторію або відкрийте GitHub issue без прикладів приватного тексту.

Не додавайте до issue:

- реальні паролі;
- приватні повідомлення;
- токени;
- вміст буфера обміну;
- повні заголовки приватних вікон.

## Принципи проєкту

- Немає телеметрії.
- Немає хмарної обробки.
- Немає історії введення.
- Немає історії буфера обміну.
- Немає автокорекції та фонової заміни тексту.
