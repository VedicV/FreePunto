# FreePunto

FreePunto — локальна macOS-утиліта для ручного перемикання і перетворення тексту між англійською, російською та українською розкладками.

FreePunto is a local macOS menu bar utility for manually converting selected text or the previous word between English, Russian, and Ukrainian keyboard layouts. It does not autocorrect text, store typing history, or send text over the network.

## Передісторія

Я давно користувався в роботі Punto Switcher, бо часто доводилося працювати різними мовами й постійно перемикатися між ними. Наприклад: у Google Sheets формули російською, JS-код англійською, листування українською.

Але мене завжди насторожувало, що це продукт Яндекса. Хто знає, що він пише, читає чи відправляє. Про всяк випадок я блокував будь-який трафік, але зрештою вирішив зробити власну перемикалку: саме таку, як мені потрібно, і таку, як я звик користуватися.

Головна вимога проста: мінімально й безпечно. Нічого не пишеться, не відправляється і не аналізується у фоні. FreePunto працює тільки тоді, коли ви явно наказали змінити останнє слово або виділений текст.

Користуйтеся, модифікуйте під себе, діліться.

## Що робимо

Застосунок працює як мінімальна безпечна заміна потрібної частини Punto Switcher, без автокорекції та зайвих функцій:

- за гарячою клавішею перетворює виділений текст;
- якщо виділення немає, перетворює останнє слово перед курсором;
- за замовчуванням основна дія викликається одиночним `Control`;
- можна налаштувати іншу комбінацію клавіш;
- після перетворення перемикає системну розкладку macOS на відповідну мову;
- перемикає текст між англійською, російською та українською розкладками;
- змінює регістр: нижній регістр, регістр речення, великі перші літери слів;
- виконує звичну фонетичну транслітерацію між кирилицею та латиницею;
- тимчасово призупиняється;
- показує в рядку меню маленьку іконку наступної мови або стан паузи.

## Чого не робимо зараз

- Жодної автокорекції.
- Жодних словників і вгадування мови.
- Жодного надсилання тексту в мережу.
- Жодної історії введення.
- Жодних користувацьких винятків для застосунків на першому етапі.
- Жодного окремого власного відкату: має працювати системний `Cmd+Z` цільового застосунку або повернення до початкового варіанта повторним перетворенням по колу.
- Жодної кросплатформності: тільки macOS.

## Розробка

Проєкт реалізовано як Swift Package:

- `PuntoCore` - чисте ядро перетворень і unit-тести.
- `FreePunto` - AppKit menu bar застосунок.

Команди:

```bash
swift build
swift test
swift run FreePunto
```

Якщо активні Command Line Tools не містять робочий XCTest, використовуйте Xcode toolchain:

```bash
DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test
```

Для збирання локального `.app` bundle:

```bash
./scripts/build_app.sh
open dist/FreePunto.app
```

Для збирання інсталяційного `.dmg`:

```bash
./scripts/build_dmg.sh
open dist/FreePunto-0.1.1.dmg
```

Для глобальних гарячих клавіш, читання і заміни тексту macOS може вимагати дозволи Accessibility та Input Monitoring.
