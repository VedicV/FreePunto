# FreePunto

macOS-приложение для ручного переключения и преобразования текста между английской, русской и украинской раскладками.

## Что делаем

Приложение работает как минимальный безопасный аналог нужной части Punto Switcher, без автокоррекции и лишних функций:

- по горячей клавише преобразует выделенный текст;
- если выделения нет, преобразует последнее слово перед курсором;
- по умолчанию основное действие вызывается одиночным `Control`;
- можно настроить другую комбинацию клавиш;
- после преобразования переключает системную раскладку macOS на соответствующий язык;
- переключает текст между английской, русской и украинской раскладками;
- меняет регистр: нижний регистр, регистр предложения, заглавные первые буквы слов;
- делает привычную фонетическую транслитерацию между кириллицей и латиницей;
- временно приостанавливается;
- показывает в строке меню маленькую иконку следующего языка или состояние паузы.

## Что не делаем сейчас

- Никакой автокоррекции.
- Никаких словарей и угадывания языка.
- Никакой отправки текста в сеть.
- Никакой истории ввода.
- Никаких пользовательских исключений приложений на первом этапе.
- Никакого отдельного собственного отката: должно работать системное `Cmd+Z` приложения, либо возврат к исходному варианту через повторное преобразование по кругу.
- Никакой кроссплатформенности: только macOS.

## Разработка

Проект реализован как Swift Package:

- `PuntoCore` - чистое ядро преобразований и unit-тесты.
- `FreePunto` - AppKit menu bar приложение.

Команды:

```bash
swift build
swift test
swift run FreePunto
```

Если активные Command Line Tools не содержат рабочий XCTest, используйте Xcode toolchain:

```bash
DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test
```

Для сборки локального `.app` bundle:

```bash
./scripts/build_app.sh
open dist/FreePunto.app
```

Для сборки установочного `.dmg`:

```bash
./scripts/build_dmg.sh
open dist/FreePunto-0.1.1.dmg
```

Для глобальных горячих клавиш, чтения и замены текста macOS может потребовать разрешения Accessibility и Input Monitoring.
