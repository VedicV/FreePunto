# Завантаження FreePunto

## Пряме завантаження DMG

[Завантажити FreePunto-0.1.1.dmg](https://github.com/VedicV/FreePunto/releases/download/v0.1.1/FreePunto-0.1.1.dmg)

Це посилання веде на файл DMG у GitHub Releases. Воно почне працювати після публікації релізу `v0.1.1` і завантаження asset-файлу `FreePunto-0.1.1.dmg`.

## Встановлення

1. Завантажте DMG.
2. Відкрийте `FreePunto-0.1.1.dmg`.
3. Перетягніть `FreePunto.app` до `Applications`.
4. Запустіть FreePunto з `Applications`.

Для глобальних гарячих клавіш, читання і заміни тексту macOS може вимагати дозволи Accessibility та Input Monitoring.

## Локальне збирання DMG

Якщо потрібно зібрати DMG локально:

```bash
./scripts/build_dmg.sh
```

Після збирання файл буде тут:

```text
dist/FreePunto-0.1.1.dmg
```
