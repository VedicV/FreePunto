# FreePunto v0.1.1

## Українською

Перший публічний open source реліз FreePunto.

FreePunto — локальна macOS-утиліта в рядку меню для ручного перетворення виділеного тексту або останнього слова між англійською, російською та українською розкладками. Застосунок не робить автокорекцію, не зберігає історію введення і не надсилає текст у мережу.

### Що всередині

- Перетворення розкладки EN/RU/UA.
- Одиночний `Control` як основна гаряча клавіша.
- Перетворення виділеного тексту або останнього слова перед курсором.
- Регістр речення, нижній регістр, title case і виправлення Caps Lock.
- Фонетична транслітерація між кирилицею та латиницею.
- Меню українською, російською або англійською.
- MIT License.

### Завантаження

Завантажте `FreePunto-0.1.1.dmg` з assets цього релізу.

### Важливо

Застосунку можуть знадобитися дозволи macOS Accessibility та Input Monitoring для глобальних гарячих клавіш, читання виділеного тексту і заміни тексту.

## English

The first public open source release of FreePunto.

FreePunto is a local macOS menu bar utility for manually converting selected text or the previous word between English, Russian, and Ukrainian keyboard layouts. It does not autocorrect text, store typing history, or send text over the network.

### Included

- EN/RU/UA keyboard layout conversion.
- Single `Control` as the default main hotkey.
- Conversion of selected text or the previous word before the cursor.
- Sentence case, lowercase, title case, and Caps Lock normalization.
- Phonetic transliteration between Cyrillic and Latin text.
- Ukrainian, Russian, and English menu localization.
- MIT License.

### Download

Download `FreePunto-0.1.1.dmg` from this release's assets.

### Important

The app may need macOS Accessibility and Input Monitoring permissions for global hotkeys, reading selected text, and replacing text.
