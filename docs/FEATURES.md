# Список функций

## Объем первой версии

Приложение: только macOS, только ручные действия, утилита в строке меню.

Любое преобразование происходит только после команды пользователя. Автокоррекции нет.

## Приоритет 0 - базовое поведение

- Глобальная команда основного преобразования.
- По умолчанию: одиночный `Control`.
- Возможность выбрать другую комбинацию клавиш.
- Преобразование выделенного текста.
- Преобразование последнего слова, если выделения нет.
- Переключение системной раскладки macOS на язык результата.
- Временная пауза.
- Минимальное меню в строке меню.
- Статусная иконка.

## Приоритет 1 - преобразование раскладки

- Английская раскладка -> русская.
- Английская раскладка -> украинская.
- Русская раскладка -> английская.
- Русская раскладка -> украинская через общую карту физических клавиш.
- Украинская раскладка -> английская.
- Украинская раскладка -> русская через общую карту физических клавиш.
- Сохранение регистра букв.
- Сохранение пробелов, переносов строк и пунктуации.
- Поддержка многострочного выделения.
- Переключение активной раскладки macOS после преобразования.

## Приоритет 2 - режимы переключения языков

### Последовательный режим

- Английский -> русский -> украинский -> английский.
- Русский -> английский -> украинский -> русский.
- Украинский -> английский -> русский -> украинский.
- Повторное нажатие на том же фрагменте перебирает следующий вариант.
- Возврат к исходнику делается повторными нажатиями по кругу.

### Режим фиксированной цели

- Русский и украинский -> английский.
- Английский -> русский или украинский, в зависимости от настройки.
- Настройка целевого языка для английского текста: русский или украинский.
- После преобразования системная раскладка переключается на язык результата.

## Приоритет 3 - регистр

- Приведение к нижнему регистру.
- Регистр предложения.
- Заглавные первые буквы слов.
- Исправление текста, случайно набранного с Caps Lock, в обычный нижний регистр.
- Сохранение пунктуации.
- Сохранение границ выделения.

Отдельный режим "все буквы заглавные" в первую версию не входит.

## Приоритет 4 - транслитерация

- Русская кириллица -> латиница.
- Украинская кириллица -> латиница.
- Латиница -> русская кириллица.
- Латиница -> украинская кириллица.
- Привычная фонетическая транслитерация, а не официальный паспортный стандарт.
- Настройка целевого кириллического языка при направлении из латиницы.
- Отдельная горячая клавиша, чтобы не путать транслитерацию с переключением раскладки.

## Приоритет 5 - меню и настройки

### Меню в строке меню

- Включено / пауза.
- Преобразовать раскладку.
- Изменить регистр.
- Транслитерировать.
- Режим: последовательный / фиксированная цель.
- Целевой язык фиксированного режима: русский / украинский.
- Целевой язык транслитерации из латиницы: русский / украинский.
- Горячие клавиши.
- Запускать при входе в систему.
- Выход.

### Настройки

- Горячая клавиша основного действия.
- Горячая клавиша изменения регистра.
- Горячая клавиша транслитерации.
- Горячая клавиша паузы.
- Режим переключения.
- Целевой язык фиксированного режима.
- Целевой язык транслитерации.
- Запуск при входе в систему.

## Приоритет 6 - статусная иконка

- Показывает следующую цель преобразования: `EN`, `RU`, `UA`.
- Показывает состояние паузы.
- По возможности показывает режим: последовательный или фиксированный.

Если точное определение языка текущего текста усложнит первую версию, иконка может показывать настройку режима и предполагаемую цель, а не пытаться постоянно анализировать текст.

## Не входит в первую версию

- Автокоррекция.
- Фоновое угадывание языка.
- Словари.
- Пользовательские исключения приложений.
- Облако и синхронизация.
- Телеметрия.
- История буфера обмена.
- История введенного текста.
- Собственный откат.
- Плагины.
- Поддержка Windows и Linux.
