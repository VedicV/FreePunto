# Список функцій

## Обсяг першої версії

Застосунок: тільки macOS, тільки ручні дії, утиліта в рядку меню.

Будь-яке перетворення відбувається тільки після команди користувача. Автокорекції немає.

## Пріоритет 0 - базова поведінка

- Глобальна команда основного перетворення.
- За замовчуванням: одиночний `Control`.
- Можливість вибрати іншу комбінацію клавіш.
- Перетворення виділеного тексту.
- Перетворення останнього слова, якщо виділення немає.
- Перемикання системної розкладки macOS на мову результату.
- Тимчасова пауза.
- Мінімальне меню в рядку меню.
- Статусна іконка.

## Пріоритет 1 - перетворення розкладки

- Англійська розкладка -> російська.
- Англійська розкладка -> українська.
- Російська розкладка -> англійська.
- Російська розкладка -> українська через спільну карту фізичних клавіш.
- Українська розкладка -> англійська.
- Українська розкладка -> російська через спільну карту фізичних клавіш.
- Збереження регістру літер.
- Збереження пробілів, переносів рядків і пунктуації.
- Підтримка багаторядкового виділення.
- Перемикання активної розкладки macOS після перетворення.

## Пріоритет 2 - режими перемикання мов

### Послідовний режим

- Англійська -> російська -> українська -> англійська.
- Російська -> англійська -> українська -> російська.
- Українська -> англійська -> російська -> українська.
- Повторне натискання на тому самому фрагменті перебирає наступний варіант.
- Повернення до початкового тексту робиться повторними натисканнями по колу.

### Режим фіксованої цілі

- Російська й українська -> англійська.
- Англійська -> російська або українська, залежно від налаштування.
- Налаштування цільової мови для англійського тексту: російська або українська.
- Після перетворення системна розкладка перемикається на мову результату.

## Пріоритет 3 - регістр

- Переведення до нижнього регістру.
- Регістр речення.
- Великі перші літери слів.
- Виправлення тексту, випадково набраного з Caps Lock, у звичайний нижній регістр.
- Збереження пунктуації.
- Збереження меж виділення.

Окремий режим "усі літери великі" до першої версії не входить.

## Пріоритет 4 - транслітерація

- Російська кирилиця -> латиниця.
- Українська кирилиця -> латиниця.
- Латиниця -> російська кирилиця.
- Латиниця -> українська кирилиця.
- Звична фонетична транслітерація, а не офіційний паспортний стандарт.
- Налаштування цільової кириличної мови для напряму з латиниці.
- Окрема гаряча клавіша, щоб не змішувати транслітерацію з перемиканням розкладки.

## Пріоритет 5 - меню та налаштування

### Меню в рядку меню

- Увімкнено / пауза.
- Перетворити розкладку.
- Змінити регістр.
- Транслітерувати.
- Режим: послідовний / фіксована ціль.
- Цільова мова фіксованого режиму: російська / українська.
- Цільова мова транслітерації з латиниці: російська / українська.
- Гарячі клавіші.
- Запускати при вході в систему.
- Вихід.

### Налаштування

- Гаряча клавіша основної дії.
- Гаряча клавіша зміни регістру.
- Гаряча клавіша транслітерації.
- Гаряча клавіша паузи.
- Режим перемикання.
- Цільова мова фіксованого режиму.
- Цільова мова транслітерації.
- Запуск при вході в систему.

## Пріоритет 6 - статусна іконка

- Показує наступну ціль перетворення: `EN`, `RU`, `UA`.
- Показує стан паузи.
- За можливості показує режим: послідовний або фіксований.

Якщо точне визначення мови поточного тексту ускладнить першу версію, іконка може показувати налаштування режиму й очікувану ціль, а не намагатися постійно аналізувати текст.

## Не входить до першої версії

- Автокорекція.
- Фонове вгадування мови.
- Словники.
- Користувацькі винятки для застосунків.
- Хмара та синхронізація.
- Телеметрія.
- Історія буфера обміну.
- Історія введеного тексту.
- Власний відкат.
- Плагіни.
- Підтримка Windows і Linux.
