# Установка и использование FreePunto

FreePunto - локальное macOS-приложение в строке меню для ручного преобразования текста между английской, русской и украинской раскладками. Приложение не делает автокоррекцию: оно работает только после вашей команды.

## Требования

- macOS 14 или новее.
- Xcode или Command Line Tools со Swift toolchain.
- Добавленные в macOS раскладки, которые вы хотите использовать: English, Russian, Ukrainian.

## Сборка приложения

Из корня проекта выполните:

```bash
./scripts/build_app.sh
```

Скрипт собирает release-версию Swift Package и создает локальный bundle:

```text
dist/FreePunto.app
```

Для разработки можно также запускать приложение без `.app` bundle:

```bash
swift run FreePunto
```

Но для проверки поведения как обычного приложения удобнее использовать `dist/FreePunto.app`.

## Запуск

После сборки откройте приложение:

```bash
open dist/FreePunto.app
```

FreePunto не открывает обычное окно. После запуска оно появляется в строке меню macOS как маленькая узкая иконка-клавиша:

- в нижнем левом углу показан символ основной клавиши `⌃`;
- в верхнем правом углу показан следующий язык `EN`, `RU` или `UA`;
- если приложение на паузе, в верхнем правом углу показан значок паузы из двух вертикальных полос, как на магнитофоне.

При наведении курсора macOS показывает текстовую подсказку вида `FreePunto: RU` или `FreePunto: PAUSE`.

## Сборка DMG

Чтобы собрать стандартный установочный `.dmg`, выполните:

```bash
./scripts/build_dmg.sh
```

Итоговый файл:

```text
dist/FreePunto-0.1.1.dmg
```

Внутри DMG лежит `FreePunto.app` и ярлык `Applications`. Откройте DMG и перетащите `FreePunto.app` на `Applications`.

## Установка в Applications

Для постоянного использования можно перенести собранное приложение в пользовательскую папку Applications:

```bash
mkdir -p ~/Applications
cp -R dist/FreePunto.app ~/Applications/FreePunto.app
open ~/Applications/FreePunto.app
```

Если вы пересобрали приложение, закройте FreePunto, замените старый `~/Applications/FreePunto.app` новым и запустите снова.

## Разрешения macOS

Для глобальных горячих клавиш, чтения выделенного текста и замены текста macOS может потребовать разрешения:

- Accessibility.
- Input Monitoring.

Откройте `System Settings -> Privacy & Security` и добавьте FreePunto в:

- `Accessibility`;
- `Input Monitoring`, если горячие клавиши или одиночный `Control` не срабатывают.

После изменения разрешений перезапустите FreePunto. В меню приложения также есть пункт `Разрешения...`, который показывает текущую подсказку по разрешениям.

## Unsigned / not notarized

Локальная сборка из `./scripts/build_app.sh` не подписана и не нотаризована. Это нормально для разработки, но macOS может заблокировать первый запуск.

Если macOS сообщает, что приложение нельзя открыть, попробуйте:

1. Открыть `System Settings -> Privacy & Security` и разрешить запуск заблокированного приложения.
2. Запустить FreePunto еще раз.

Если приложение было скачано или перенесено между машинами и macOS держит quarantine-метку, для локальной разработки можно снять ее командой:

```bash
xattr -dr com.apple.quarantine dist/FreePunto.app
```

Используйте эту команду только для сборки, которой доверяете.

## Меню в строке меню

Нажмите на иконку FreePunto в строке меню. По умолчанию меню на русском, но язык можно поменять в пункте `Язык меню`.

Основные пункты меню:

- `Пауза` / `Продолжить` - временно приостановить или включить приложение.
- `Сменить раскладку (Control)` - преобразовать раскладку выделенного текста или последнего слова.
- `Изменить регистр (Control+Option+C)` - изменить регистр по выбранному режиму.
- `Транслитерация (Control+Option+T)` - выполнить транслитерацию.
- `Режим переключения` - выбрать режим по кругу или фиксированную цель.
- `Цель раскладки` - выбрать фиксированную цель `RU` или `UA`.
- `Цель транслитерации` - выбрать целевой кириллический язык `RU` или `UA` для транслитерации из латиницы.
- `Регистр` - выбрать режим регистра.
- `Язык меню` - выбрать русский, украинский или английский интерфейс.
- `Горячие клавиши и настройки...` - открыть настройки.
- `Запускать при входе` / `Не запускать при входе` - включить или выключить запуск при входе в систему.
- `Разрешения...` - показать подсказку по системным разрешениям.
- `Выйти из FreePunto` - закрыть приложение.

## Горячие клавиши

По умолчанию:

- Основное преобразование раскладки: одиночный `Control`.
- Изменение регистра: `Control+Option+C`.
- Транслитерация: `Control+Option+T`.
- Пауза: `Control+Option+P`.

Главная клавиша смены раскладки - именно `Control`. В настройках она показана отдельной строкой `Смена раскладки клавиатуры`. Там можно:

- увидеть текущее значение, по умолчанию `Control`;
- нажать `Записать` и задать другую комбинацию;
- нажать `Вернуть Control`, чтобы снова использовать одиночный `Control`.

Для остальных действий также можно нажать `Записать` рядом с нужной строкой и задать свою комбинацию.

### Одиночный Control

Одиночный `Control` - основной триггер по умолчанию. Нажмите и отпустите левый или правый `Control` без других клавиш, чтобы выполнить `Сменить раскладку`.

Если такой триггер мешает вашим привычным сочетаниям или не работает в конкретной среде:

1. Откройте `Горячие клавиши и настройки...`.
2. В строке `Смена раскладки клавиатуры` нажмите `Записать`.
3. Нажмите новую комбинацию.

## Как пользоваться преобразованием

Обычный сценарий:

1. Выделите текст в любом приложении.
2. Нажмите одиночный `Control` или выбранную горячую клавишу.
3. FreePunto заменит выделение преобразованным текстом.
4. Если возможно, FreePunto переключит системную раскладку macOS на язык результата.

Если текст не выделен, FreePunto пытается выбрать и преобразовать последнее слово перед курсором.

Преобразование работает локально. FreePunto не отправляет текст в сеть и не хранит историю ввода.

## Режимы раскладки

### По кругу

Режим `По кругу` перебирает языки по кругу. Повторное действие на том же фрагменте дает следующий вариант.

Примеры циклов:

- английский -> русский -> украинский -> английский;
- русский -> английский -> украинский -> русский;
- украинский -> английский -> русский -> украинский.

Этот режим удобен, когда нужно быстро исправлять текст и иногда возвращаться к предыдущему варианту повторным нажатием.

### Фиксированная цель

Режим `Фиксированная цель` использует фиксированную цель:

- русский и украинский текст преобразуется в английский;
- английский текст преобразуется в `RU` или `UA`, в зависимости от `Цель раскладки`.

В строке меню фиксированный режим помечается маленькой точкой на языковой иконке.

## Регистр

Действие `Изменить регистр` применяет режим из меню `Регистр`:

- `Нижний регистр` - все буквы маленькие.
- `Регистр предложения` - первая буква предложения заглавная.
- `Первые буквы слов` - первые буквы слов заглавные.
- `Исправить Caps Lock` - привести текст, набранный как будто с Caps Lock, к обычному нижнему регистру.

Режим можно выбрать в меню `Регистр` или в окне `Горячие клавиши и настройки...`.

## Транслитерация

Действие `Транслитерация` преобразует текст между кириллицей и латиницей:

- русская кириллица -> латиница;
- украинская кириллица -> латиница;
- латиница -> русская или украинская кириллица.

Для направления из латиницы выберите цель в меню `Цель транслитерации`: `RU` или `UA`.

Транслитерация использует привычное фонетическое соответствие, а не официальный паспортный стандарт.

## Пауза

Пауза временно отключает преобразования. Ее можно включить через:

- пункт меню `Пауза`;
- горячую клавишу паузы;
- чекбокс `Включено` в `Горячие клавиши и настройки...`.

Когда приложение на паузе, в строке меню показан значок паузы из двух вертикальных полос. Горячая клавиша паузы продолжает работать, чтобы можно было быстро включить FreePunto обратно.

## Запуск при входе в систему

Чтобы запускать FreePunto автоматически при входе в macOS:

1. Откройте меню FreePunto в строке меню.
2. Нажмите `Запускать при входе`.

Отключение делается тем же пунктом, который после включения называется `Не запускать при входе`.

Если macOS не разрешает включить автозапуск, FreePunto покажет ошибку. Проверьте системные настройки входа в учетную запись и попробуйте снова.

## Troubleshooting

### FreePunto не появляется в Dock

Это ожидаемо. FreePunto - menu bar приложение. Ищите маленькую иконку `EN`, `RU`, `UA` или значок паузы в строке меню macOS.

### Горячие клавиши не работают

Проверьте:

- FreePunto запущен и не находится в состоянии паузы.
- FreePunto добавлен в `Accessibility`.
- FreePunto добавлен в `Input Monitoring`.
- После выдачи разрешений приложение было перезапущено.
- Другая программа не перехватывает ту же горячую клавишу.

Если одиночный `Control` не подходит, задайте явную комбинацию в `Горячие клавиши и настройки...`.

### Текст не заменяется

Проверьте:

- есть ли выделенный текст;
- находится ли курсор в обычном текстовом поле;
- не защищено ли поле системными ограничениями, например поле пароля;
- разрешен ли FreePunto в `Accessibility`.

Некоторые приложения могут не отдавать выделенный текст через Accessibility. В таком случае FreePunto пытается работать через выделение последнего слова и буфер обмена, но это тоже может быть ограничено конкретным приложением.

### После преобразования не переключается раскладка

Добавьте нужную раскладку в `System Settings -> Keyboard -> Text Input -> Input Sources`. FreePunto может переключиться только на доступный в системе input source.

### Появляется ошибка `Горячие клавиши недоступны`

Обычно это означает, что macOS не дала создать глобальный event tap. Выдайте разрешения `Accessibility` и `Input Monitoring`, затем полностью закройте и снова запустите FreePunto.

### macOS не открывает приложение

Локальная сборка не подписана и не нотаризована. Разрешите запуск в `System Settings -> Privacy & Security` или, для доверенной локальной сборки, снимите quarantine-метку:

```bash
xattr -dr com.apple.quarantine dist/FreePunto.app
```

### Запуск при входе не включается

Проверьте, что вы запускаете именно `.app` bundle из `dist/FreePunto.app`, а не только `swift run FreePunto`. Для проверки автозапуска поведение `.app` ближе к реальному использованию.
