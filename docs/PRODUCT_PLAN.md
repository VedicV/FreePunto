# Продуктовый план

## Как я понял продукт

Нужно сделать macOS-приложение в строке меню, которое не исправляет текст автоматически, а выполняет только ручные команды пользователя.

Главная команда: взять выделенный текст, а если выделения нет - последнее слово перед курсором, и преобразовать его одним из способов:

- сменить текст между английской, русской и украинской клавиатурными раскладками;
- после преобразования переключить активную системную раскладку macOS на язык результата;
- изменить регистр;
- выполнить фонетическую транслитерацию между кириллицей и латиницей;
- временно приостановить приложение.

Принцип первой версии: только нужные функции, без автокоррекции, словарей, угадывания и фоновой отправки текста.

## Базовое поведение

### Область действия

1. Если есть выделенный текст - преобразуется выделение.
2. Если выделения нет - преобразуется последнее слово перед курсором.
3. Приложение не запускает автокоррекцию само.
4. Приложение не хранит историю ввода.
5. Приложение не отправляет текст в сеть.

### Основная клавиша

По умолчанию основное действие вызывается одиночным нажатием `Control`.

Также можно разрешить обычные комбинации клавиш, если пользователь захочет заменить одиночную клавишу.

Для macOS надо отдельно проверить:

- одиночное нажатие `Control`;
- левый и правый `Control`;
- конфликт с системными сочетаниями клавиш.

Если одиночный `Control` в каком-то окружении будет мешать системным сочетаниям, пользователь сможет выбрать явную комбинацию.

## Переключение языков

Нужно два режима.

### Последовательный режим

Повторные нажатия перебирают варианты по кругу.

Если исходный текст английский:

```text
Английский -> русский -> украинский -> английский
```

Если исходный текст русский:

```text
Русский -> английский -> украинский -> русский
```

Если исходный текст украинский:

```text
Украинский -> английский -> русский -> украинский
```

Это также заменяет отдельный собственный откат: если нужно вернуть исходный вариант, пользователь может нажимать основную клавишу дальше до возврата по кругу. Где возможно, системное `Cmd+Z` целевого приложения тоже должно работать нативно.

### Режим фиксированной цели

Более предсказуемый режим:

- русский и украинский текст преобразуются в английский;
- английский текст преобразуется в выбранный язык: русский или украинский.

После преобразования приложение переключает системную раскладку macOS на язык результата.

## Функции первой версии

- macOS-приложение в строке меню.
- Глобальная горячая клавиша основного преобразования.
- Одиночный `Control` по умолчанию.
- Настройка горячих клавиш.
- Преобразование выделенного текста.
- Преобразование последнего слова, если выделения нет.
- Перекладка текста между английской, русской и украинской раскладками по таблице физических клавиш.
- Переключение системной раскладки macOS на язык результата.
- Последовательный режим.
- Режим фиксированной цели.
- Горячая клавиша для изменения регистра.
- Горячая клавиша для транслитерации.
- Временная пауза.
- Минимальное меню.
- Минимальные настройки.
- Иконка в строке меню, показывающая следующий язык или паузу.

## Регистр

Нужна отдельная команда для преобразования регистра:

- нижний регистр - все буквы маленькие;
- регистр предложения - первая буква предложения заглавная, остальные обычные;
- заглавные первые буквы слов.

Отдельный режим "все буквы заглавные" сейчас не нужен.

Команда должна сохранять:

- пробелы;
- переносы строк;
- пунктуацию;
- границы выделенного текста.

## Транслитерация

Нужна отдельная команда для привычной фонетической транслитерации:

- русский текст кириллицей -> латиница;
- украинский текст кириллицей -> латиница;
- латиница -> русский текст кириллицей;
- латиница -> украинский текст кириллицей.

Для направления `латиница -> кириллица` нужен выбор целевого языка: русский или украинский.

Важно не смешивать транслитерацию с переключением раскладки:

- переключение раскладки: `ghbdtn` -> `привет`;
- транслитерация: `привет` -> `privet`.

## Минимальное меню

Меню в строке меню должно содержать только нужное:

- включено / пауза;
- выполнить основное преобразование;
- изменить регистр;
- транслитерировать;
- режим переключения: последовательный или фиксированная цель;
- целевой язык для фиксированного режима: русский или украинский;
- целевой язык для транслитерации из латиницы: русский или украинский;
- настройки горячих клавиш;
- запускать при входе в систему;
- выход.

## Иконка

Иконка должна показывать:

- `EN`, `RU`, `UA` - следующий язык для основного преобразования;
- состояние паузы;
- по возможности маленький признак режима: последовательный или фиксированный.

Если надежно определять язык текущего фрагмента слишком сложно, первая версия может показывать текущую настройку режима и предполагаемую следующую цель.

## Что не входит в первую версию

- Автокоррекция.
- Словари.
- Определение языка через ML, AI или n-граммы.
- Подсказки и всплывающие исправления.
- Исключения приложений.
- Собственная история действий.
- Собственный механизм отката.
- Синхронизация настроек.
- Поддержка Windows и Linux.
- Расширения и плагины.
- История введенного текста.

## Этапы

### Этап 0 - проверка возможностей macOS

- Проверить одиночный `Control` как глобальный триггер.
- Проверить чтение выделенного текста через Accessibility.
- Проверить запасной способ через буфер обмена с обязательным восстановлением прежнего содержимого.
- Проверить замену последнего слова без выделения.
- Проверить переключение системной раскладки macOS через Text Input Source APIs.
- Проверить, как операции попадают в системный стек отмены целевого приложения.

### Этап 1 - первая версия для выделенного текста

- Приложение в строке меню.
- Настройки горячих клавиш.
- Преобразование выделенного текста между EN/RU/UA.
- Последовательный режим и режим фиксированной цели.
- Переключение системной раскладки после преобразования.
- Пауза.
- Статусная иконка.

### Этап 2 - последнее слово

- Определение последнего слова перед курсором.
- Замена последнего слова без повреждения окружающего текста.
- Возврат по кругу через повторное преобразование.
- Тесты в TextEdit, Notes, Safari/Chrome, Messages/Telegram, редакторе кода.

### Этап 3 - регистр и транслитерация

- Нижний регистр.
- Регистр предложения.
- Заглавные первые буквы слов.
- Фонетическая транслитерация RU/UA <-> EN.
- Настройки направления транслитерации.
- Тесты на смешанном тексте, пунктуации и многострочных выделениях.

### Этап 4 - полировка и релиз

- Первичный экран настройки разрешений macOS.
- Code signing и notarization.
- Минимальная диагностика без сохранения текста.
- Упаковка релизной сборки.
